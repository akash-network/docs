## akash query tx

Query for a transaction by hash, "<addr>/<seq>" combination or comma-separated signatures in a committed block

### Synopsis

Example:
$ akash query tx <hash>
$ akash query tx --type=acc_seq <addr>/<sequence>
$ akash query tx --type=signature <sig1_base64>,<sig2_base64...>
